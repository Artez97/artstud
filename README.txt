# Календарное приложение

Простое календарное приложение, разработанное с использованием Nuxt3, Vue.js и Pinia.

## Установка

1. Убедитесь, что у вас установлен Node.js. Можно загрузить его с [официального сайта Node.js](https://nodejs.org/).

2. Клонируйте репозиторий на локальный компьютер:

```bash
git clone https://github.com/yourusername/calendar-app.git


## Перейдите в каталог с проектом:
cd calendar-app

## Установите зависимости:
npm install

## Запуск
После установки зависимостей вы можете запустить проект с помощью следующей команды:

bash
Copy code
npm run dev
Откройте браузер и перейдите по адресу http://localhost:3000, чтобы увидеть работающее приложение.
